Initial Settings

When you are setting Calypso's boot options you should be really careful on syntax because syntax is very important.

You must put space before and after "=".

And, fallow the instruction carefully.

Instructions:

  1. If you want Calypso to start on every boot. Set the "Start Calypso at every computer start up" setting in "bootsettings.txt" file to Yes. The "bootsettings.txt" file is in bin folder.

    2. Ex: Start Calypso at every computer start up = Yes

  2. Set the "Location to start Calypso on every start up" setting to an absolute path, that refers to the Calypso's root directory/folder using python syntax, like in the example.

    3. Ex: Location to start Calypso on every start up = E:\\work_space\\Calypso(latest-version)

  3. If you want to prevent Calypso from starting up on every boot. First, press Windows key + R, type shell:startup and, be sure the directory/folder that got open doesn't include calypso.py, becuase in windows that's the directory that windows looks up on boot to start applications. Finally, set "Start Calypso at every computer start up" to No.

    4. Ex: Start Calypso at every computer start up = No

Setting Up The Requirements

Setting up the requirements of calypso is pretty easy. All you have to do is fallow the fallowing steps:

  1. Install python programming language from it's official website. Don't forget to add python to path during installation.
  2. Open the project's root folder(calypso_1.00) and double click on setup.py
  3. Install espeak to it's location. Don't forget to install it French language by adding "fr" without double quotes in Select which voices to install section.
  4. Install graphviz to it's location. Be sure you that you check marked the option to add graphviz to the path.

Skip the fallowing optional steps if you don't want calypso to start at every boot.

Optional steps:

  1. Press Windows Key + R
  2. Type shell:startup
  3. Copy paste the calypso.py to the location that got opened.
